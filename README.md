# ComedyBOT

## Requirements

- [Node(v6.4) and npm](http://nodejs.org)

## Installation

1. `git clone git@github.com:shy2net/ComedyBOT.git`
2. `cd ComedyBOT`
3. `npm install`
3. `npm start`
4. Go to `http://localhost:3000`

## Usage

After running `npm start` and going into `http://localhost:3000`, simply play with the bot!
