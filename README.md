# ComedyBOT

## Requirements

- [Node(v6.4) and npm](http://nodejs.org)

## Installation

1. `git clone git@github.com:shy2net/node-skeleton.git`
2. `cd node-skeleton`
3. `npm install`
3. `npm start`
4. Go to `http://localhost:3000`
5. Enjoy life.

## Usage

After running `npm start`, simple play with the bot!
